# Personal Developer Portfolio Website

A clean, responsive, and modern personal portfolio website built with HTML, CSS, and vanilla JavaScript to showcase projects, skills, and professional experience.

## Description

This project is a fully-functional, single-page portfolio designed to create a strong online presence for developers. It features a sleek interface, smooth animations, and dedicated sections for an introduction, skills, project showcase, and a functional contact form. The goal is to provide a professional and visually appealing platform to impress potential employers and collaborators.

## Usage

Once you open `index.html` in your browser, you can navigate the single-page application using the navigation bar links:

-   **Home**: The landing section with a brief introduction.
-   **About**: A detailed bio, personal information, and a CV download button.
-   **Skills**: A dynamic display of technical skills with animated progress bars that activate on scroll.
-   **Portfolio**: A grid showcasing recent projects with links to the live versions and source code.
-   **Contact**: A fully functional contact form that sends messages directly to your inbox via Formspree.

Simply scroll through the page or use the navigation links to jump between sections.

## Features

-   **Fully Responsive Design**: Looks great on all devices, from mobile phones to desktop monitors.
-   **Dynamic Animations**: Subtle fade-in effects and skill bar animations powered by the Intersection Observer API.
-   **Project Showcase**: An interactive grid to display your work with overlay links to GitHub and live demos.
-   **Functional Contact Form**: An integrated, easy-to-configure contact form using Formspree.
-   **CV Download**: A dedicated button allowing visitors to download your resume directly.
-   **Clean & Modern UI**: A minimalist and professional design focused on readability and user experience.
